About This Website

OS Installation Tutorials is an educational resource created as part of the Operating Systems course. This website provides curated video tutorials and written descriptions for installing and configuring Windows operating systems, Windows Server editions, and core infrastructure roles commonly used in IT environments.

Whether you're a student learning about operating system architectures, an IT professional setting up legacy systems, or a tech enthusiast exploring computing history, these tutorials offer clear, visual guidance on how each platform is installed and what its main features are. The focus is on understanding the purpose and capabilities of each operating system or server, not just memorizing step-by-step screens.

Beyond installation, the Activities section guides you through hands-on tasks such as working with product keys, creating multiple user accounts, adding server roles like DHCP, configuring RAID, setting up file services, hosting applications and web sites, and managing environments through Group Policy. These activities are designed to simulate real-world administration tasks in a safe lab environment.
