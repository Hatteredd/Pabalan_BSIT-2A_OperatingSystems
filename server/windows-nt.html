Windows NT Server Installation

1993

About Windows NT Server

Windows NT Server (version 3.1) was Microsoft's first 32-bit server operating system, built on the NT kernel. It introduced NTFS, domains, and enterprise networking capabilities. NT Server laid the foundation for all future Windows Server editions.

Basic Installation Steps

  • Boot from Windows NT Server installation media (floppy + CD)
  • Insert Setup Boot Disk 1 and boot the system
  • Insert Setup Disk 2 and Disk 3 when prompted
  • Insert the NT Server CD when requested
  • Choose Express or Custom setup
  • Select or create partition and choose NTFS or FAT
  • Enter installation directory (typically C:\WINNT)
  • Setup copies files and restarts
  • Configure computer name and server role
  • Create Administrator account and password
  • Configure network adapter and protocols
  • Join or create domain/workgroup
  • Complete setup and restart
