Windows Server 2008 Installation

2008

About Windows Server 2008

Windows Server 2008 introduced Hyper-V virtualization, Server Core (minimal installation), and improved Active Directory. It shared the Vista kernel and brought significant advances in security, reliability, and management. Server Manager replaced Add/Remove Roles.

Basic Installation Steps

  • Boot from Windows Server 2008 DVD
  • Press any key to boot from DVD
  • Select language and region‚Äîclick Install now
  • Choose edition: Full Installation or Server Core
  • Accept the license terms
  • Choose upgrade or custom installation
  • Select partition and drive options (NTFS recommended)
  • Setup copies files and restarts
  • Set Administrator password (required)
  • Initial Configuration Tasks (ICT) appears
  • Set computer name and join domain/workgroup
  • Configure network adapter and IP settings
  • Configure Windows Update
  • Add roles and features via Server Manager
  • Setup completes; server ready for configuration
