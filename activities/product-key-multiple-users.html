Product Key and Multiple Users

Activity Overview

This activity focuses on how Windows activation works and how product keys relate to the number of users or devices you are allowed to run the software on. In a lab setting, it is important to understand the difference between retail keys, OEM keys, evaluation keys, and volume licensing keys, because each type has its own limitations and intended usage.

You will examine license terms, identify where to enter the product key during setup or post-installation activation, and discuss scenarios such as sharing a single key across many machines versus using unique keys for each system. The main goal is for you to understand that activation is tied to licensing rules and hardware, and that adding multiple users to a computer or domain does not mean you can ignore license limits.

By the end of this activity, you should be comfortable explaining how one Windows license can legally be used, how activation status affects functionality, and how multiple local or domain users can share the same installed operating system while still respecting Microsoft licensing guidelines.
