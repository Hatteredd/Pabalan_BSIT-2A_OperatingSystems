Application & Web Server

Activity Overview

This activity covers deploying web and application services on a Windows server, using Internet Information Services (IIS) or a similar platform. You will see how to install the web server role, configure basic sites and application pools, and place simple web applications or static pages under the correct directories and permissions.

The goal is to understand how a Windows server transitions from being a basic file or domain controller into a platform that can host intranet sites, lab applications, and web-based tools. You will think about service accounts, bindings, ports, and security, and how these pieces come together to support applications used by multiple users in your Operating Systems course.
