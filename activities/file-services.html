Activity Overview

File services are at the heart of most Windows servers. In this activity, you will design and configure shared folders that users can access over the network, assign NTFS and share permissions, and think about how to separate data for different groups such as students, staff, or departments. Properly configured file services allow people to collaborate while keeping sensitive data secure.

You will also consider topics such as home folders, roaming profiles, offline files, and quotas. The goal is to understand how Windows manages data on disk, how permissions are evaluated, and how a single file server can safely host data for many users and classes in an Operating Systems laboratory environment.
