Windows 98 Installation

1998

About Windows 98

Windows 98 built upon Windows 95 with improved Internet Explorer, USB support, and FAT32 file system. It featured Active Desktop and better integration with web content, making it popular for home and office use.

Basic Installation Steps

  • Boot from Windows 98 CD or run SETUP.EXE from within DOS
  • Run Scandisk to check for disk errors before installation
  • Start the Setup wizard and accept the license agreement
  • Enter the 25-character product key
  • Select the installation directory
  • Choose installation option: Typical, Compact, Portable, or Custom
  • Enter computer and user information
  • Setup detects hardware‚Äîhave drivers ready if needed
  • Create a Windows 98 startup disk when prompted
  • Restart to complete installation
