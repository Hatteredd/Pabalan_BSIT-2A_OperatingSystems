Windows 8.1 Installation

2013

About Windows 8.1

Windows 8.1 refined the touch-focused design of Windows 8 by bringing back the Start button, improving the Start screen, and making it easier to boot directly to the desktop. It tried to balance the needs of tablet users with those of traditional keyboard-and-mouse desktop users.

In the video you will see how Windows 8.1 is installed, how to navigate between the Start screen and desktop, and how some of the new apps and settings pages behave. This helps you understand the design experiments that eventually led to the more traditional interface that returned in Windows 10.
